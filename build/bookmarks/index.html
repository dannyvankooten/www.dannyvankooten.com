Software

I've been happily running Linux on my laptop since 2016.

Made possible by the following list of mostly open-source software (not exhaustive).

  • Debian, the Linux distribution I use as my daily driver.
  • Gnome, desktop environment.
  • Firefox, web browser.
  • uBlock Origin, ad-blocker.
  • Bitwarden, password manager.
  • Bash, sh compatibe shell. I like Fish too, but it's just too convenient to find Bash pre-installed pretty much anywhere.
  • Imagemagick, image manipulation.
  • Todoist, to-do app that runs my life (proprietary).
  • GIMP, image editor.
  • Inkscape, SVG editor.

Web development

  • PHP It's getting much better lately and still powers the vast majority of web backends.
  • Symfony, PHP web framework and components.
  • Nginx, web server software.
  • PostgreSQL, MariaDB, or SQLite. SQL database servers.
  • MDN documentation, all the HTML, CSS and JS documentation you need.
  • Doctrine, ORM and Database Abstraction Layer for PHP.
  • PHPStorm, IDE (proprietary).
  • Zola, static site generator (in Rust, so compiles to a single binary).
  • Jekyll, alternative static site generator with lots of plugins. Requires Ruby though.
  • WordPress, content management system. Sadly, most themes are bloated.
  • A11Y accessibility checklist
  • Sustainable Web Manifesto
  • sourcehut, git hosting service.

I strive to not use any CSS frameworks and only use JavaScript sparingly. Have also been intrigued by #NoBuild lately.
